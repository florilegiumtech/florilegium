# florilegium
Florilegium – Botânica Tecnológica desenvolve cosméticos naturais e fitoativos a partir de plantas nativas das Missões (RS), unindo conhecimento tradicional, química verde e extração com solventes eutéticos profundos (DES). Projeto dedicado à inovação em fitocosméticos, sustentabilidade e ciência aplicada à biodiversidade brasileira.
